# Руководство по использованию веб-интерфейса CrossPoster

## Обзор интерфейса

CrossPoster предоставляет удобный веб-интерфейс для управления аккаунтами социальных сетей и кросспостингом. Интерфейс включает в себя:

1. **Главная страница** - общая информация и тестовый репост
2. **Страница аккаунтов** - подключение и настройка аккаунтов социальных сетей
3. **Страница настроек** - конфигурация приложения

## Подключение аккаунтов

### 1. ВКонтакте

1. Перейдите на страницу аккаунтов
2. Нажмите "Настроить" под иконкой ВКонтакте
3. Введите API-токен ВКонтакте:
   - Создайте standalone-приложение на https://vk.com/apps?act=manage
   - Получите токен с необходимыми правами доступа
4. При необходимости укажите ID группы для публикации
5. Нажмите "Сохранить"

### 2. Telegram

1. Найдите @BotFather в Telegram
2. Создайте нового бота командой `/newbot`
3. Скопируйте токен, который @BotFather выдал вам
4. Введите токен в поле "Токен Telegram-бота"
5. Укажите канал для публикации (в формате @channel_name или числовой ID)
6. Нажмите "Сохранить"

### 3. Instagram

1. В настоящее время Instagram требует использование официального API
2. Введите логин и пароль от аккаунта Instagram
3. Нажмите "Сохранить" (пароль будет зашифрован)

### 4. Pinterest

1. Зарегистрируйтесь как разработчик на https://developers.pinterest.com/
2. Создайте приложение и получите API-ключ
3. Введите API-ключ в соответствующее поле
4. Укажите доску для публикации в формате username/board_name
5. Нажмите "Сохранить"

### 5. YouTube

1. Создайте проект в Google Cloud Console
2. Включите YouTube Data API v3
3. Создайте учетные данные OAuth 2.0
4. Введите полученный токен в поле "OAuth токен YouTube"
5. Укажите ID канала
6. Нажмите "Сохранить"

## Тестирование репоста

После подключения аккаунтов вы можете протестировать функционал кросспостинга:

1. Перейдите на главную страницу
2. Введите текст тестового поста в соответствующее поле
3. При необходимости добавьте медиафайлы
4. Нажмите "Отправить тестовый репост"
5. Система автоматически опубликует пост во все подключенные и активные аккаунты

## Безопасность

- Все API-ключи хранятся в зашифрованном виде в базе данных
- Ключи невозможно увидеть в веб-интерфейсе после сохранения
- Для дополнительной безопасности используйте SSL-соединение
- Регулярно обновляйте токены доступа

## Мониторинг

На главной странице отображается статистика:
- Общее количество постов
- Количество репостов
- Количество активных аккаунтов
- Время последней синхронизации

## Устранение неполадок

### Аккаунт не подключается
- Проверьте правильность введенных токенов
- Убедитесь, что у токена есть необходимые права доступа
- Проверьте подключение к интернету

### Пост не публикуется
- Убедитесь, что аккаунт активен
- Проверьте ограничения платформы на публикации
- Проверьте формат медиафайлов

### Ошибка "Произошла ошибка при сохранении аккаунта"
- Проверьте формат вводимых данных
- Убедитесь, что токен имеет достаточные права
- Попробуйте снова через некоторое время

## Поддержка

Если у вас возникли проблемы с использованием веб-интерфейса:
1. Проверьте логи приложения
2. Убедитесь, что все зависимости установлены правильно
3. Проверьте конфигурацию .env файла
4. Обратитесь к документации по конкретной социальной сети
